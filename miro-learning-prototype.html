Block 1 — The Miro canvas as a learning space
Live · Facilitator demo + guided practice · Whole group
45 min
Key features covered
  • Frames — creating zones per activity
  • Sticky notes, text boxes, basic shapes
  • Connectors and simple arrows
  • Inviting participants + live cursors
  • Presentation mode (follow the facilitator)
Practice task
  • Build a 3-frame board structure
  • Label each frame for a distinct activity
  • Add a welcome sticky in frame 1
  • Try presentation mode on their board
Pedagogical framing
  • Frames = lesson structure made visible
  • The board mirrors a workshop agenda
  • Visual space acts as shared working memory
  • Spatial design signals learning intent
Facilitation strategy
  • Think aloud while building live
  • Show the "messy" version then the clean one
  • Use the pre-work stickies as a live example
Facilitator guidance Start by reviewing the pre-work stickies together — this validates their effort and anchors the session in their own context. Build the demo board in real time rather than showing a finished version. Narrate each decision: "I'm creating a frame here because I want learners to know exactly where this activity lives." Give 10 minutes of free practice before moving on.

Block 2 — Running a collaborative activity in Miro
Live · Trainer-as-learner simulation · Pairs then whole group
60 min
Key features covered
  • Voting with dot stickers
  • Timer widget for timed tasks
  • Grouping and clustering sticky notes
  • Locking objects for stability
  • Zoom to selection for whole-group review
Simulation activity
  • Topic: "What makes a good training activity?"
  • Diverge: everyone adds stickies (5 min)
  • Cluster: pairs group similar ideas (10 min)
  • Converge: dot-vote on top insights (5 min)
  • Debrief from both learner + facilitator view
Collaborative learning link
  • Diverge → converge as a learning structure
  • Accountability through visible contribution
  • Facilitation moves: signal phases, hold silence
  • Managing "Miro chaos" in live sessions
Discussion prompts
  • What did it feel like as a participant?
  • What would you do differently as a facilitator?
  • Where did the tool help — and where did it hinder?
Facilitator guidance Position this as a simulation: participants play learners first, then step into the facilitator perspective for the debrief. When clustering happens, expect noise and some disorder — that is intentional. In the debrief, highlight what Miro made easier AND harder. This tension is the pedagogical insight you want them to hold.

Block 3 — Workshop design clinic
Live · Peer review + facilitator coaching · Pairs then group
45 min
New features introduced
  • Comments and @mentions for feedback
  • Breakout frames (assigning groups to zones)
  • Share board link + export snapshot
  • Board permissions (viewer vs editor)
Clinic process
  • Pairs swap board links
  • Leave 2 "warm" + 1 "wonder" comment each
  • Facilitator selects 2 boards to discuss live
  • Whole group: what design choices stand out?
Workshop design principles
  • Structuring a workshop arc on Miro
  • Facilitator view vs participant view
  • The 3-zone workshop frame pattern
  • When NOT to use Miro in a session
Peer learning strategy
  • "Warm / Wonder" feedback protocol
  • Feedback as formative assessment
  • Safe-to-fail design review culture
Facilitator guidance Use comments rather than verbal feedback in the first 15 minutes — this keeps it in Miro and models the tool. When you select boards for group discussion, choose one that's spatial and well-organised, and one that's text-heavy. The contrast surfaces a pedagogical insight without you having to state it directly. End with: "What would a learner's first 90 seconds on this board feel like?"

Block 4 — Micro-teach: run your mini-session
Live · 5-min facilitation rehearsal per trainer · Rotating
45 min
What each trainer does
  • Facilitates 5 min of their mapped lesson
  • Others join the board as participants
  • Rotate every 5 min (2–3 trainers max)
  • 3-min debrief after each: warm / wonder / next
Observation focus
  • Did the frame structure help orientation?
  • How did the trainer give instructions?
  • What went wrong — and how did they recover?
  • What would they do differently next time?
Deliberate practice
  • Low-stakes rehearsal builds confidence fast
  • Immediate peer debrief = formative assessment
  • Watching peers models new facilitation moves
  • Failure here is the safest it will ever be
What to name in the debrief
  • Moments of "productive confusion"
  • How spatial design guided (or confused) learners
  • The tool fading into the background
Facilitator guidance Keep time strictly — 5 minutes is hard to fill and easy to overrun. Let the trainer lead; resist jumping in unless they freeze. During the debrief, ask the participants before asking the trainer: "What was it like to be a learner on that board?" Only select 2 trainers if time is tight — depth of debrief matters more than number of micro-teaches.
Each trainer has facilitated a real Miro session — not just watched one
